1. PROJECT GOAL
==================================================

Build an AI Vet feature with:

1. A Python FastAPI backend
2. Gemini API integration using Google AI Studio API key

5. Public deployment so the BD website can call the backend

4. BACKEND FLOW
==================================================

Step 1:
Create FastAPI app in main.py

Used:
- app = FastAPI()

Step 2:
Load environment variables

Used:
- from dotenv import load_dotenv
- load_dotenv()
- os.getenv("GEMINI_API_KEY")

Step 3:
Define request body model

Used:
class VetRequest(BaseModel):
    pet_type: str
    breed: str
    age: str
    weight: str
    city: str
    symptoms: str

Step 4:
Create health route

Used:
GET /ping

Response:
{"status": "ok"}

Step 5:
Create AI route

Used:
POST /ai-vet

Input:
- pet_type
- breed
- age
- weight
- city
- symptoms

Step 6:
Build prompt using submitted pet data

Step 7:
Call Gemini API using google-genai

Used:
from google import genai
client = genai.Client(api_key=os.getenv("GEMINI_API_KEY"))
response = client.models.generate_content(
    model="gemini-2.5-flash",
    contents=[{"role": "user", "parts": [{"text": prompt}]}],
    config={"response_mime_type": "application/json"},
)

Step 8:
Parse Gemini response as JSON

Used:
json.loads(response.text)

Step 9:
Return parsed JSON back to frontend

Step 10:
Handle bad/missing API key or invalid JSON with HTTPException

==================================================
9. COMMON LOCAL ISSUES FACED
==================================================

Issue 1:
Yellow import warnings in VS Code

Cause:
- VS Code using wrong Python interpreter

Fix:
- Select interpreter:
  `C:\gemini api project\venv\Scripts\python.exe`

Issue 2:
Port 8000 already in use

Error:
- WinError 10013

Cause:
- Another Python/Uvicorn process already listening on port 8000

Fix options:
```powershell
Stop-Process -Id <PID>
```

or run on another port:
```powershell
uvicorn main:app --reload --port 8001
```

Issue 3:
`GEMINI_API_KEY is not set`

Cause:
- `.env` empty or server not restarted

Fix:
- Put key in `.env`
- Restart Uvicorn

==================================================
11. IMPORTANT FRONTEND LESSON
==================================================

Wrong:
Using BD page URL as API endpoint

Example wrong URL:
```text
https://pettinutservices.directoryup.com/vet-ai
```

Why wrong:
- This is a webpage
- It returns HTML
- AJAX expects JSON
- Result becomes messy page source or "Not Found"

Correct:
Use deployed FastAPI endpoint

Correct URL:
```text
https://pettinut-ai-vet.onrender.com/ai-vet
```
